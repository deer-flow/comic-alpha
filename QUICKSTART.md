# 🚀 快速开始 - 小红书内容生成功能

## 5分钟快速体验

### 第一步：启动后端服务

```bash
# 进入项目目录
cd comic_alpha

# 启动后端（如果还没启动）
python backend/app.py
```

你应该看到：
```
* Running on http://127.0.0.1:5003
```

### 第二步：打开前端页面

在浏览器中打开 `index.html` 文件，或者：

```bash
# 使用Python启动简单的HTTP服务器
python -m http.server 8000
```

然后访问：`http://localhost:8000`

### 第三步：配置API

1. 点击页面右上角的 **⚙️ 配置** 按钮
2. 输入你的配置：
   - Base URL: `https://api.openai.com/v1`（或你的自定义URL）
   - Model: 选择 `gpt-4o-mini`（推荐）
3. 点击 **💾 保存配置**

### 第四步：生成漫画

1. 在 "输入你的 OpenAI API Key" 框中输入你的API Key
2. 在提示框中输入漫画描述，例如：
   ```
   生成一个关于小明学习编程的故事，要有趣味性
   ```
3. 设置生成页数：`3`
4. 选择漫画风格：`哆啦A梦风格`
5. 选择语言：`中文`
6. 点击 **✨ AI 生成多页分镜**

等待几秒钟，漫画就生成好了！

### 第五步：生成小红书内容

1. 漫画生成完成后，你会看到一个红色的按钮：**📱 生成小红书内容**
2. 点击这个按钮
3. 等待几秒钟
4. 弹出窗口显示生成的内容：
   - 📝 标题
   - 📄 正文
   - 🏷️ 标签

### 第六步：复制和使用

1. 点击 **📋 复制全部** 按钮
2. 打开小红书APP
3. 创建新笔记
4. 粘贴内容
5. 上传你的漫画图片
6. 发布！🎉

## 常见问题快速解决

### ❌ 按钮不显示
**原因**：漫画还没生成
**解决**：先完成第四步，生成漫画

### ❌ 生成失败
**原因**：API Key无效或网络问题
**解决**：
1. 检查API Key是否正确
2. 确认网络连接
3. 查看浏览器控制台错误信息

### ❌ 后端连接失败
**原因**：后端服务未启动
**解决**：
```bash
python backend/app.py
```

### ❌ 复制功能不工作
**原因**：浏览器权限问题
**解决**：
1. 允许浏览器访问剪贴板
2. 或手动选择文本复制

## 进阶技巧

### 技巧1：调整生成风格
在生成漫画时选择不同的风格，会影响小红书内容的描述：
- 哆啦A梦风格 → 温馨可爱的文案
- 美式漫画风格 → 热血激昂的文案
- 水彩风格 → 文艺清新的文案

### 技巧2：多次生成
如果对第一次生成的内容不满意：
1. 再次点击生成按钮
2. 每次生成的内容都会有所不同
3. 选择你最喜欢的版本

### 技巧3：手动优化
生成的内容可以作为基础：
1. 复制到文本编辑器
2. 根据个人风格调整
3. 添加个性化内容
4. 调整emoji使用

### 技巧4：批量生成
如果你有多个漫画项目：
1. 为每个项目生成内容
2. 保存到文档中
3. 建立自己的文案库
4. 后续可以参考和复用
